# Bank Loan Default Risk Analysis Using Power BI

## Project Overview
This project focuses on analyzing bank loan data to identify factors that influence loan default risk.
An interactive Power BI dashboard was developed to support data-driven decision making in banking.

## Dataset
- Source: Kaggle (Bank Loan Data)
- Features include customer age, income, credit score, loan amount, loan intent, and loan status.

## Tools & Technologies
- Power BI
- DAX
- Power Query
- CSV Dataset

## Dashboard Features
- Total Loans Issued
- Loan Default Rate (%)
- Loan Status Distribution
- Income vs Loan Default Analysis
- Credit Score vs Loan Default Analysis
- Loan Intent vs Default Analysis
- Interactive slicers (Gender, Home Ownership, Previous Defaults)

## Key Insights
- Customers with low credit scores have higher default rates
- High loan amount relative to income increases default risk
- Borrowers with previous loan defaults are the highest risk group
- Certain loan purposes show higher default frequency

## Outcome
This analysis helps banks identify risky borrowers and improve loan approval strategies.
